# WardReady — Current Project State
Last updated: 2026-06-27

## Overall Status

WardReady is a production-ready offline-first Flutter application for Ethiopian medical education. The codebase demonstrates mature architecture with Riverpod state management, Drift SQLite local database with FTS5 search, Supabase backend integration, and comprehensive error handling. Core flows (auth, article browsing, quiz with SM-2 spaced repetition, progress tracking) are implemented and functional. The app uses Material 3 design with Navy/Gold theming and includes onboarding, legal screens, admin dashboard, and subscription gating.

## Architecture

- **Framework:** Flutter with Dart SDK ^3.11.0
- **State:** Riverpod (flutter_riverpod, riverpod_annotation)
- **Local DB:** Drift (SQLite) with FTS5 full-text search
- **Backend:** Supabase (Auth, PostgreSQL, Storage)
- **Routing:** GoRouter (declarative, route-based)
- **Design System:** Material 3, Navy (#1A237E) + Gold (#F9A825) theme, dark mode default
